# 📊 Sales Analytics Dashboard

A comprehensive sales analytics dashboard built using Microsoft Excel 
to analyze and visualize business sales data effectively.


---

## 📌 About The Project
This project was built to demonstrate practical data analysis skills 
using Excel. It analyzes real-world style sales data to uncover trends, 
customer behavior patterns, and order channel performance.

---

## ✨ Features
- 📈 Sales trend analysis across months and quarters
- 👥 Customer behavior tracking and segmentation  
- 🛒 Order channel performance comparison
- 🗺️ Regional sales breakdown
- 💰 Revenue and profit margin calculations
- 📊 Dynamic pivot tables and interactive charts
- 🎯 KPI tracking with visual indicators

---

## 🛠️ Tools & Technologies
| Tool | Usage |
|------|-------|
| Microsoft Excel | Main platform |
| Pivot Tables | Data summarization |
| Pivot Charts | Visualizations |
| VLOOKUP / INDEX-MATCH | Data lookup |
| SUMIF / COUNTIF | Calculations |
| Conditional Formatting | Visual indicators |
| Data Validation | Input control |

## 🚀 How To Use
1. Download the repository
2. Open `Sales_Dashboard.xlsx` in Microsoft Excel
3. Enable editing if prompted
4. Use slicers and filters to explore the data
5. All charts update automatically
